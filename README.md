# Challenge Sprint 4 - Programa de Bolsas Compass.uol

## Sobre o projeto

O projeto consiste no desenvolvimento de exercícios disponibilizados pelo Programa de Bolsas da Compass.uol, neles foram desenvolvidos códigos utilizando operadores e manipulação de arquivos em Python.

## Instalação

Para executar o projeto, será necessário instalar os seguintes programas:

- [Python](https://realpython.com/installing-python/)
- [Pandas](https://www.pythoncentral.io/how-to-install-pandas-in-python/)
